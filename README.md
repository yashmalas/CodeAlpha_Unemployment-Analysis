# Unemployment Analysis with Python

## Project Overview

This project analyzes unemployment trends in India using Python and data visualization techniques.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Dataset

The dataset contains unemployment statistics from different regions of India.

## Analysis Performed

* Data Cleaning
* Data Exploration
* State-wise Unemployment Analysis
* Monthly Trend Analysis
* Urban vs Rural Comparison

## Visualizations

1. Top 10 States with Highest Unemployment Rate
2. Monthly Unemployment Trend
3. Urban vs Rural Unemployment Comparison

## Files Included

* Unemployment in India.csv

## Conclusion

This project helps understand unemployment patterns across different regions and time periods in India using data analysis and visualization.
